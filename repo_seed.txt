# chromostereopsis — REPO SEED

## The Goldmine

Red floats. Blue sinks. Your own eyeball does the 3D. No glasses. This is not anaglyph — there are no colored glasses, no shutter sync, no VR headset. Just pure saturated red and pure saturated blue, painted flat, and your own eye's chromatic aberration splits them into depth. Some people see red forward, some see blue forward. The effect is subtle, uncanny, and completely real. It's also the glasses-free cousin of `chromadepth` — cross-link them hard.

## The Math

### Longitudinal Chromatic Aberration
The human eye refracts different wavelengths by different amounts:
- Red (700nm) → focused *behind* the retina → appears to advance.
- Blue (400nm) → focused *in front* of the retina → appears to recede.
- The brain interprets this focal mismatch as depth.

### Pupil Decentration
The effect is stronger when the pupil is not perfectly centered over the optical axis — which is true for most people. The combination of chromatic aberration + pupil decentration creates the depth illusion.

### Polarity Flip
On a **white** background, the effect can **invert** (blue advances, red recedes). This is because the eye adapts differently to bright vs dark fields.

## The GLSL / JS Pipeline

### Pass Order
1. **Scene** → assign depth/layer to each element.
2. **Map** depth → red↔blue axis: near = pure `#FF0000`, far = pure `#0000FF`.
3. **Render** max-saturation pure-channel colors on controlled background.
4. **Optional** CA enhancement pass (`chromatic_aberration` repo).
5. **Minimal post** — the effect lives in the eye. Heavy finishers kill it.

### Red-Blue Field Generator
```glsl
uniform float u_depth; // 0.0 = near (red), 1.0 = far (blue)
uniform vec3 u_bg;     // background color

in vec2 v_uv;
out vec4 fragColor;

void main() {
  // Pure red near, pure blue far
  vec3 color = mix(vec3(1.0,0.0,0.0), vec3(0.0,0.0,1.0), u_depth);
  // Ensure max saturation
  float maxC = max(color.r, max(color.g, color.b));
  if (maxC > 0.0) color /= maxC;
  fragColor = vec4(color, 1.0);
}
```

### Depth-from-Hue Mapper
```glsl
// Map scene depth to red-blue axis
vec3 depthToChromostereopsis(float depth) {
  // depth: 0.0 = closest, 1.0 = farthest
  return mix(vec3(1.0,0.0,0.0), vec3(0.0,0.0,1.0), depth);
}
```

## Aesthetic Post + Tonal Guide

### Deep Field
Starfield. Red stars in foreground, blue stars in background, black ground. The depth pops without any stereo trick.

### Floating Glyphs
Red sigils (runes, symbols) hovering off a blue plane. The glyphs feel like they're floating in front of the screen.

### Polarity Demo
Same art rendered on black vs white backgrounds, side by side. Watch the depth invert. Educational and eerie.

### Breathing Layers
Slow red↔blue cycle. The layers pulse in depth, breathing. Hypnotic, subtle.

### Type Pops
Red text floating off a blue background. The text feels embossed, 3D, without any shadow or bevel.
