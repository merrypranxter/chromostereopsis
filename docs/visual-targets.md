# Visual Targets

## 1. Deep Field
Starfield on black. Red stars in foreground, blue stars in background. The depth pops without any stereo trick. Subtle but real.

## 2. Floating Glyphs
Red sigils hovering off a blue plane. The glyphs feel like they're floating in front of the screen. Runes, symbols, abstract marks.

## 3. Polarity Demo
Same pattern on black vs white backgrounds, side by side. The depth inverts between halves. Educational and eerie. Proves the effect is in the eye.

## 4. Breathing Layers
Slow red↔blue cycle. The layers pulse in depth, breathing. Hypnotic, subtle, alive.

## 5. Type Pops
Red text floating off a blue background. The text feels embossed, 3D, without any shadow or bevel. Pure color doing the work.
